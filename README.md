#Rock paper Scissors Game-Python
A simple Command-line Rock Paper Scissors game built using python,where the user plays against the computer.
##Concepts Used:
  --> Python basics
  --> user input
  --> random module
  --> if-elif-else condition
##How Game Works:
   -The user selects:
        '0'=>Rock
        '1'=>Paper
        '2'=>Scissors
  -The computer randomly selects one option>
  -The winner is decided based on standard Rock Paper Scissors rules.
